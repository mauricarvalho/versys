Renewal Process

For a renewal process, there are some factors that need to be considered in order to determine whether tests are applicable or if we can proceed with the renewal only with the documentation. They are the following:
1 - If the product has been modified since the last certification or not;
2 - If ANATEL changed the rules during the period;
3 - If the product presented any non-conformity in the initial certification that became necessary to modify the product in order to attend the regulation;
4 - If it is applicable the Resolution 529 (Safety Testing). If so, it is required to repeat this test in the renewal, even if the answer to the previous items is “No”.

Documentation needed when the product has not suffered changes:
• Renewal Letter (we can provide you a template)
• Current external pictures
• Current internal pictures
• Test reports (it will be confirmed if tests are needed or not after we check the 04 items listed previously)
• ISO 9001 2008 Certificate for the factory (when applicable)

Documentation needed when the product has suffered changes:
• Renewal Letter (we can provide you a template)
• Current external pictures
• Current internal pictures
• ISO 9001 2008 Certificate for the factory (when applicable)
• Test reports (it will be confirmed if tests are needed or not after we check the 04 items listed previously)
• Product Manual (in case it has been updated)
• Technical specification (in case it has been updated)
• Electric scheme
